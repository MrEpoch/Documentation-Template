## Vytvoření

Jako první si vytvoříme v /app routě složku `loading.tsx/jsx`, to bude složka které poběží když se něco načítá.
Pokud jí umístíme do podsložky např. `/app/login/loading.tsx` , tak ten loader uvidí jen `/app/login` routa

### Můj obsah složky

```tsx
"use client";

export default function Loading() {
  return (
    <div className="min-h-screen w-full z-50 flex items-center justify-center top-0 right-0">
      <div className="loader"></div>
    </div>
  );
}
```

### Dotatečné CSS

**CSS loader vyberu z [css loaders](https://css-loaders.com/)**

```css
.loader {
  width: 80px;
  height: 40px;
  color: #000;
  border: 2px solid currentColor;
  border-right-color: transparent;
  padding: 3px;
  background: repeating-linear-gradient(
      90deg,
      currentColor 0 10px,
      #0000 0 15px
    )
    0/0% no-repeat content-box content-box;
  position: relative;
  box-sizing: border-box;
  animation: l5 2s infinite steps(6);
}
.loader::before {
  content: "";
  position: absolute;
  top: -2px;
  bottom: -2px;
  left: 100%;
  width: 10px;
  background:
    linear-gradient(
        #0000 calc(50% - 7px),
        currentColor 0 calc(50% - 5px),
        #0000 0 calc(50% + 5px),
        currentColor 0 calc(50% + 7px),
        #0000 0
      )
      left / 100% 100%,
    linear-gradient(
        currentColor calc(50% - 5px),
        #0000 0 calc(50% + 5px),
        currentColor 0
      )
      left / 2px 100%,
    linear-gradient(
        #0000 calc(50% - 5px),
        currentColor 0 calc(50% + 5px),
        #0000 0
      )
      right/2px 100%;
  background-repeat: no-repeat;
}
@keyframes l5 {
  100% {
    background-size: 120%;
  }
}
```
